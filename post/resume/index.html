負責專案

iAP2

iAP2 (iPod Accessory Protocol v2) 是蘋果公司規範的協定，用來與 iDevice 溝通。我負責在公司的新系統實作 iAP2，讓公司產品能夠與 iPod/iPhone/iPad 溝通。

Transaction count testing app

開發命令列與 Android app，模擬正常的交易行為。用來記錄在電池充飽電的情況下，可以成功執行幾次交易，以及產品能夠運作多久時間。
